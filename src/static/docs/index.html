Wio

Wio is a backend-as-a-service platform for developing web applications with AI agents. It provides a managed database, real-time websockets, LLM API, markdown rendering, sound playback, site cookies, and instant deployment — all accessible through a lightweight client SDK.

The development workflow is designed to be fully AI-driven. You scaffold a project with the CLI, point an AI coding agent at it, and deploy with a single command. No server configuration, no build pipelines, no infrastructure management.

How It Works​

  1. Scaffold a project with wio init my-site
  2. Build your app using the wio.js SDK (let an AI agent write the code)
  3. Deploy with wio push — your site is instantly live at https://my-site.wio.onl

Your app's HTML, CSS, and JS files are uploaded and served from Wio's infrastructure. The SDK (wio.js) is automatically available at /wio.js and provides access to all platform services.

Platform Services​

ServiceSDK MethodDescription
Databasewio.useRelation()Managed PostgreSQL with a query builder interface
WebSocketswio.wsReal-time communication via Socket.IO rooms
LLM APIwio.ask()Server-side AI text generation
Markdownwio.renderMarkdown()Server-side markdown to HTML conversion
Sound Playerwio.playSound()Broadcast sound effects to all connected users
Cookieswio.cookiesSite-scoped cookie storage via the SDK

Tech Stack​

Wio is built with TypeScript on Bun and Fastify. Data is stored in PostgreSQL, static files in MinIO (S3-compatible), and real-time features use Socket.IO.
